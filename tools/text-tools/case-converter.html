Case Converter

Convert text between different case formats easily and quickly

Usage Tips
  • Enter or paste your text in the input field
  • Select your desired case format from the options above
  • Use the Copy button to copy the converted text
  • Use the Swap button to exchange input and output text
  • Use the Clear button to reset both fields
Common Use Cases
  • UPPERCASE: Headers, Acronyms
  • lowercase: URLs, File names
  • Title Case: Titles, Headings
  • Sentence case: Paragraphs, Articles
  • camelCase: JavaScript variables
  • PascalCase: Class names, Components
  • snake_case: Python variables, CSS classes
  • kebab-case: URLs, CSS classes
